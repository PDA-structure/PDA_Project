   - If `!moved && Math.hypot(dx, dy) > 3`, set `moved = true;` AND `card.classList.add('dragging')` (CSS uses this for `cursor: grabbing`).

     - Get card dimensions: `const w = card.offsetWidth, h = card.offsetHeight;`
     - Viewport bounds in layer-local coords:
       - `minLeft = -layerRect.left + 40 - w;`  // card right edge ≥ 40 from viewport left
       - Wait — re-derive: we want ≥40 px of the card visible on each viewport edge. In viewport-px, the card's left edge in viewport-space is `layerRect.left + left`. Constraints:
         - `(layerRect.left + left) + w >= 40` → `left >= 40 - w - layerRect.left`  (card's right edge ≥ 40 from viewport left, i.e., at least 40 px sticks in)
         - `(layerRect.left + left) <= window.innerWidth - 40` → `left <= window.innerWidth - 40 - layerRect.left`  (card's left edge ≤ 40 from viewport right)
         - Same for top: `top >= 40 - h - layerRect.top`, `top <= window.innerHeight - 40 - layerRect.top`
     - Apply: `card.style.left = Math.max(minLeft, Math.min(maxLeft, left)) + 'px';` (same for top).

   - `card.classList.remove('dragging')`.
