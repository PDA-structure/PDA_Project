  <name>Task 3: Drag handlers with threshold and viewport clamp</name>

  <action>
Add the drag handler function to ui/truss2d/script.js, in the floating-panels section (between dockCard and the DOMContentLoaded registration).

`function onCardDragStart(e, section)`:
- Guard: return if `section.dataset.state !== 'floating'`
- Guard: return if `e.target.classList.contains('card-float-btn')` (don't drag when clicking the button)
- Capture `startX = e.clientX`, `startY = e.clientY`, `startLeft = parseFloat(section.style.left) || 0`, `startTop = parseFloat(section.style.top) || 0`
- `let moved = false`
- Bring-to-top immediately on mousedown: `section.style.zIndex = String(++_floatZIndex)` (per D-7)
- Define `onMove(ev)`:
  - Compute dx, dy from start position
  - If `!moved && Math.hypot(dx, dy) > 3`: set `moved = true`, add `.dragging` class to section, add `.card-dragging` class to body
  - If `!moved`: return (ignore sub-3px micro-movements -- prevents accidental drags per D-6 adaptation note)
  - `ev.preventDefault()` (suppress text selection during drag)
  - Viewport clamp per D-8: get `#cardFloatLayer` bounding rect, compute min/max left/top so at least 40px of the section remains visible on each viewport edge:
    ```
    const minLeft = 40 - w - layerRect.left;
    const maxLeft = window.innerWidth  - 40 - layerRect.left;
    const minTop  = 40 - h - layerRect.top;
    const maxTop  = window.innerHeight - 40 - layerRect.top;
    ```
  - Apply clamped position to section.style.left/top
- Define `onUp()`:
  - Remove document-level mousemove + mouseup listeners
  - If `moved`: remove `.dragging` from section, remove `.card-dragging` from body
- Register `document.addEventListener('mousemove', onMove)` and `document.addEventListener('mouseup', onUp)`

Note: Unlike frame2d, truss2d sections have no native click-to-toggle-disclosure on h3. The 3px threshold simply prevents accidental micro-drags (cleaner than frame2d's disclosure-preservation concern). No need for stopPropagation or preventDefault on the non-moved case.
