# Plan 260505-u2h-01 — Frame2D Bundle A Follow-up: Button + Card Density

**Goal:** Make toolbar cards stay in a single row at typical Mac viewport widths and reduce button text size so cards aren't dominated by oversized labels. Builds on tke commit `77e2134`.

## Tasks

### Task 1 — Apply density delta (auto)
**Files:** `ui/frame2d/style.css`

**Action:**
1. Modify the `.tool-btn` rule (around line 242-250) — add `font-size: 11px;` and change `padding` from `var(--space-2) var(--space-3)` to `var(--space-1) var(--space-2)` (which evaluates to `4px 8px` per the design token scale at lines 35-39).
2. Modify the `.card, .panel-section` shared rule (around line 184-190 from tke) — change `min-width: 140px` to `min-width: 110px`.

No other rules touched. Diff target: ~3-5 lines.
