<objective>
Add per-member cross-sectional area A to the truss2d tool, Phase 1 of truss2d member
properties (Area A only; material density / self-weight UDL is a separate later Phase 2,
explicitly OUT of scope here).

Mirror the frame2d per-member E/I/A override pattern, simplified to A only: let the user
assign A (cm²) to individual members in the UI, send A as a scalar-or-array to
`POST /solve/truss2d`, and have it feed the FEM correctly so it affects BOTH displacement
(via EA/L stiffness) AND the axial stress (force/area) shown in results.

Purpose: real trusses mix member sizes; a uniform A under/over-estimates both deflection
and member stress. This unlocks per-member sizing and is the foundation for Phase 2
self-weight.

Output: backend accepts per-member A; solver/adapter compute correct per-member stiffness +
stress; UI gains a "Member A" editor + per-member A in the solve payload and Save/Load JSON;
one analytical test proving correctness.

Key discovery (drives the small backend footprint): the truss solver
(`solvers/truss2d.py`) ALREADY accepts A as scalar OR array — `_A(e)` returns `self.A[e]`
when `self.A` is an ndarray, and both `_assemble_global_stiffness()` (`k = E*A/L`) and
`solve_member_forces()` use `_A(e)`. So per-member A flows into stiffness AND member forces
with NO solver change. The only backend gap is the API request model, which currently types
`A: float` and so rejects a list.
</objective>

Solver already handles scalar-or-array A (solver_core/.../solvers/truss2d.py):
```python
class Truss:
    def __init__(self, nodes, members, E, A, forceVector, restrainedDoF):
        ...
        self.A = np.asarray(self.A, float) if np.ndim(self.A) else float(self.A)  # list → ndarray
    def _A(self, e):
        return self.A[e] if isinstance(self.A, np.ndarray) else self.A
    # _assemble_global_stiffness: k = self._E(e) * self._A(e) / L   ← per-member stiffness
    # solve_member_forces:        k = self._E(e) * self._A(e) / L   ← per-member axial force
```

Adapter computes stress (solver_core/.../adapters/truss_adapters.py):
```python
member_forces_arr = np.array(t.mbrForces, float) if t.mbrForces is not None else None
member_stresses = member_forces_arr / self.model.A if member_forces_arr is not None else None
# meta["member_stresses"] = member_stresses.tolist()   ← already per-member when A is array
```

frame2d per-member resolution pattern to mirror (ui/frame2d/script.js solve(), A-only):
```js
const anyOverride = members.some(m => m.A_override != null);
const A = anyOverride
  ? members.map(m => (m.A_override != null ? m.A_override : A_cm2) * 1e-4)  // cm² → m²
  : A_cm2 * 1e-4;
```

frame2d save/load override persistence to mirror (object keyed by member index string):
```js
// save:  canvas.memberOverrides[String(idx)] = { A_cm2: m.A_override }
// load:  m.A_override = overrides.A_cm2 != null ? overrides.A_cm2 : null;
```

truss2d UNIT conventions (CLAUDE.md): A cm² → m² is ×1e-4. Global A input id="inputA"
(default 100 cm²). Truss tension positive. 2 DOF/node.
