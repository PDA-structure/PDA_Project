### `solver_core/.../loads/combinations.py` (NEW — generator + superpose + envelope)

**Analog:** `adapters/truss_adapters.py:9-46` — the orchestration pattern (take inputs, loop the solver, assemble a result dataclass with numpy arrays). The combination layer sits **above** the engine and calls `engine.solve(...)` per case, exactly as the adapter calls `Truss(...).solve_*()`.

**Orchestration shape to mirror** (truss_adapters.py:13-46):
```python
def solve(self) -> AnalysisResult:
    t = Truss(nodes=..., members=..., E=..., A=..., forceVector=..., restrainedDoF=...)
    t.solve_displacements()
    t.solve_member_forces()
    FG = t.solve_reactions()
    member_forces_arr = np.array(t.mbrForces, float) if t.mbrForces is not None else None
    return AnalysisResult(solver="truss2d", UG=..., FG=..., member_forces=member_forces_arr, meta={...})
```

**Linearity basis to rely on (no re-solve, D-09):** `Truss` builds a fixed `Ks` once (truss2d.py:90) and member force is linear in `UG` (truss2d.py:114: `N = k*(-c*u0 - s*u1 + c*u2 + s*u3)`). Therefore `N(Σ f_k) = Σ N(f_k)` — superpose cached per-case `member_forces`.

**Patterns to produce** (RESEARCH §"Concrete dataclasses" + §"Code Examples"):
1. Dataclasses: `LoadCase{id,name,nature,loads}`, `CombinationTerm{case_id,factor}` (frozen — factors live ONLY here, D-01), `Combination{name,cls,terms,expression,leading}`, `EnvelopeMember{member,max_tension,gov_tension,max_compression,gov_compression}`, `EnvelopeResult{members,combo_to_members}`.
2. `generate_eurocode(cases, gamma, psi0, families)` — 6.10 leading-action iteration (RESEARCH §Code Examples). Skip empty cases (Pitfall 4). Degenerate Dead+Imposed → single `1.35G+1.5Q` row (D-12).
3. `superpose(combination, per_case_forces, n_members)` → `np.zeros` + `Σ factor·per_case`.
4. `envelope(combinations, per_case_forces)` → `np.vstack` + `np.argmax`/`np.argmin` carrying governing-combo names (D-18/D-19 forward map + reverse `combo_to_members` index).

**Hard rule:** pure numpy/python; no printing, no matplotlib; never imports `visualization/`.
