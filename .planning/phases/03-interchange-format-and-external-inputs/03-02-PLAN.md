```python
"""
Integration tests for the PDA interchange format (Phase 3).

Tests verify:
1. Fixture JSON files match the canonical schema (schema_version, solver, required fields)
2. Fixture JSON is solve-ready — can be POSTed to the API and returns correct analytical results
3. Canvas state is present for visual round-trip restoration

Analytical reference cases:
  - Frame2d cantilever: E=200e9, I=1e-4, L=1, F=-10000 → Uy_tip = -FL³/(3EI) ≈ -1.667e-4 m
  - Truss2d horizontal bar: E=200e9, A=0.01, L=1, F=1000 → Ux_tip = FL/(EA) = 5e-7 m
"""

import json
import pytest
from pathlib import Path
from fastapi.testclient import TestClient
from api_server.app import app

FIXTURES = Path(__file__).parent / "fixtures"
client = TestClient(app)


@pytest.fixture
def frame2d_model():
    with open(FIXTURES / "sample_pda_frame2d.json") as f:
        return json.load(f)


@pytest.fixture
def truss2d_model():
    with open(FIXTURES / "sample_pda_truss2d.json") as f:
        return json.load(f)
```

`test_truss2d_fixture_is_valid_json(truss2d_model)`: Assert `truss2d_model["schema_version"] == "1.0"`, `truss2d_model["solver"] == "truss2d"`, required keys: `nodes`, `members`, `E`, `A`, `forceVector`, `restrainedDoF`, `canvas`.

**Solve-readiness tests:**

`test_frame2d_fixture_is_solve_ready(frame2d_model)`:
- Extract solve payload: `{k: v for k, v in frame2d_model.items() if k not in ("schema_version", "canvas")}`
- Set `payload["solver"] = "frame_v2"` (engine name, not file routing key)
- POST to `/solve/frame2d` with `json=payload`
- Assert `resp.status_code == 200`
- Parse result: `result = resp.json()`
- Assert `"UG" in result` and `"member_moments" in result`
- Analytical check: cantilever tip Uy = -FL^3/(3EI) = -10000 * 1.0**3 / (3 * 200e9 * 1e-4) = -1/60000 m approx -1.6667e-4 m
- `assert result["UG"][4] == pytest.approx(-10000.0 * 1.0**3 / (3 * 200e9 * 1e-4), rel=1e-6)`
- Note: UG index 4 is DOF 5 (1-based) = node 2 Uy (0-based index 4 in the flat array)

`test_truss2d_fixture_is_solve_ready(truss2d_model)`:
- Extract solve payload: `{k: v for k, v in truss2d_model.items() if k not in ("schema_version", "canvas")}`
- POST to `/solve/truss2d` with `json=payload`
- Assert `resp.status_code == 200`
- Analytical check: Ux at node 2 = FL/(EA) = 1000 * 1.0 / (200e9 * 0.01) = 5e-7 m
- `assert result["UG"][2] == pytest.approx(1000.0 * 1.0 / (200e9 * 0.01), rel=1e-6)`
- Note: UG index 2 is DOF 3 (1-based) = node 2 Ux
