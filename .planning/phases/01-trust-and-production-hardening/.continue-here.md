- [ ] CONSTRAINT: SFD Vj sign — `V(ξ) = Vi*(1-ξ) - Vj*ξ` (negate Vj). Same convention. Fixed in this session.
- [ ] CONSTRAINT: `beamPinRight` requires `pinDoF` — condensed theta DOF must also appear in `pinDoF`.
- [ ] CONSTRAINT: `FG = Kp @ UG` includes applied loads — equilibrium checks must use specific restrained DOF indices.

**Do not proceed until all boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| BMD/SFD wrong side | Using `Mi*(1-ξ) + Mj*ξ` for BMD — gives wrong sign when Mj≠0 (e.g. point load via intermediate node, fixed ends) | blocking | Use `Mi*(1-ξ) - Mj*ξ` for both BMD and SFD (negate j-end term) |
| sum(FG) equilibrium check | `np.sum(FG[:,0]) + load` always = 0 — FG includes applied loads | blocking | Check specific restrained DOF reactions |
