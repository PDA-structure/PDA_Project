In `frame_adapters.py` — In the `FrameV2Adapter.solve()` method, after `s.solve_structure()` and before creating `AnalysisResult`, compute member stresses. The cross-sectional area `A` is available from `self.model.A` (or `self.model.A_beam` / `self.model.A_bar` for mixed structures). For simplicity in this phase, use the effective area that the solver used:

```python
# Determine effective area for stress calculation
A_eff = self.model.A
if A_eff is None:
    A_eff = self.model.A_beam if self.model.A_beam is not None else 1.0

member_forces_arr = np.array(s.mbrForces, float) if s.mbrForces is not None else None
member_stresses = member_forces_arr / A_eff if member_forces_arr is not None else None
```

Then update the `meta` dict in the AnalysisResult constructor to include `member_stresses`:
```python
meta={
    "n_nodes": int(s.n_nodes),
    "n_members": int(s.n_members),
    "member_stresses": member_stresses.tolist() if member_stresses is not None else None,
},
```

In `truss_adapters.py` — In `Truss2DAdapter.solve()`, after computing member forces and reactions, compute stress:

```python
member_forces_arr = np.array(t.mbrForces, float) if t.mbrForces is not None else None
```

Update the meta dict:
```python
meta={

    "n_members": int(len(t.members)),
    "member_stresses": member_stresses.tolist() if member_stresses is not None else None,
},
```

Note: Stress = axial force / area (axial stress only, per D-15). Bending stress is deferred to Phase 3.
